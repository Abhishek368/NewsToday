
# News Today

Live Breaking News app which uses [NewsAPI](https://newsapi.org/) to fetch Breaking News headlines from the Rest Api.

## Components and Libraries Used

* Kotlin
* Retrofit
* Glide
* Coroutines
* ViewModel
* Room 
* Navigation 
* Architecture - MVVM
